## 🚀 À Propos

Ce projet est la page d'accueil officielle d'**IRIS Junior Entreprise**. Il a été réalisé dans le cadre d'un atelier collaboratif visant à illustrer l'application des bonnes pratiques **HTML5** (sémantique, accessibilité, validation W3C) et du **workflow collaboratif professionnel** avec Git et GitHub (branches, commits atomiques, pull requests, code review).

Il constitue un exemple concret et reproductible de comment travailler en équipe sur un projet web, du clonage jusqu'au déploiement.

---

## 🛠 Technologies Utilisées

- [HTML5](https://developer.mozilla.org/fr/docs/Web/HTML) - Structure sémantique et accessible
- [Git / GitHub](https://github.com/) - Versionnement et collaboration
- [Validateur W3C](https://validator.w3.org/) - Validation du code HTML

## ⚙️ Installation

Aucun serveur ni dépendance externe n'est requis. Suivez ces étapes :

```bash
# Cloner le dépôt
git clone https://github.com/irisjuniorenterprise/iris-homepage.git

# Accéder au dossier
cd iris-homepage
```

Ouvrez ensuite le fichier `index.html` directement dans votre navigateur.

---

## 🖥 Utilisation

Ouvrez `index.html` dans un navigateur web moderne. La page affiche :

- Un **en-tête** (`<header>`) avec le nom et le logo d'IRIS JE
- Une **navigation** (`<nav>`) vers les différentes sections
- Une **section héro** (`<main>`) de présentation
- Un **pied de page** (`<footer>`) avec les informations de contact
- *(Bonus)* Un **formulaire de contact** accessible

> ⚠️ Aucune interaction dynamique ni style CSS n'est inclus, conformément aux objectifs de l'exercice.

## 🤝 Contribution

Les contributions suivent le **GitHub Flow** :

1. Créez votre branche de fonctionnalité (`git checkout -b feature/nom-fonctionnalite`).
2. Développez avec des **commits atomiques** (un changement logique par commit).
3. Committez vos modifications (`git commit -m 'feat: description du changement'`).
4. Pushez sur la branche (`git push origin feature/nom-fonctionnalite`).
5. Ouvrez une **Pull Request** et demandez une revue (principe des 4 yeux).
6. Mergez après approbation.
